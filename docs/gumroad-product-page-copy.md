# Gumroad Product Page Copy

Paste or adapt this copy for the Gumroad product page and delivery page.

## What You Receive

- Desktop app download for AI YouTube Shorts Generator.
- Gumroad license key for activating the app on your device.
- In-app setup guide after license activation.

## Required For First Generation

You need a MuAPI key before generating your first clip. API mode is the recommended first setup because it handles download, transcription, highlight selection, and clipping through MuAPI.

## Optional Advanced Setup

Local mode is optional beta/advanced functionality in the first release. It may require an OpenAI key, a local runtime pack, local model download or repair, FFmpeg, yt-dlp, and system-specific troubleshooting. Use API mode first unless you already know you want local processing.

## First Run

1. Install and open the desktop app.
2. Activate the app with your Gumroad license key.
3. Open Settings -> Configuration -> API Providers.
4. Add your MuAPI key with Add MuAPI Profile.
5. Return to Generate, paste a YouTube URL, keep API mode selected, and run generation.
6. After generation finishes, use Open Clip, Copy Link, or Open Folder to access your output.

## Refunds And Support

Use your Gumroad purchase/support channel for support and refund requests. Refund requests are reviewed manually within 7 days from purchase. The app does not include an automated refund engine.

Device reset requests may require support review if your license is already bound to another device.

## Known Limits

- Third-party API outages, quotas, pricing, policy changes, or account issues can affect generation.
- You are responsible for rights, permissions, and platform-term compliance for source media.
- AI-selected clips and captions should be reviewed before publishing.
- Local mode is beta/advanced and may not work on every machine without runtime/model setup or repair.
- Device resets, refund review, and support responses are manual processes.
